# Date Equality Checker

This C++ program reads two dates from the user and checks if they are equal.

## Features

- Input validation for year, month, and day.
- Leap year support.
- Clear prompts and error messages.

## Example

- ``Input``
```
Date1:
Enter a Year : 2025
Enter a Month : 07
Enter a Day : 29

Date2:
Enter a Year : 2025
Enter a Month : 07
Enter a Day : 28
```
- ``Output``
```
No, Date1 Not Equal Date2.
```
