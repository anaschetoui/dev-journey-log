## <span style="color: red">Question</span>
### Are the following valid variable names and why?
- Age<br>
- My-Age<br>
- My_Age_<br>
- _Age<br>
- 2x<br>
- X2<br>
- My Name<br>
- MyName<br>
- #Name<br>
- $Name<br>
- N@me<br>
- int<br>
- float<br>

## <span style="color: green">Answers</span>
- Age -> Correct -> Letters allowed and isn't taken<br>
- My-Age -> False -> Not allowed cause there is - <br>
- My_Age_ -> Correct -> Letters and underscore allowed<br>
- _Age -> Correct -> underscore at first allowed<br>
- 2x -> False -> Numbers at first not allowed<br>
- X2 -> Correct -> Letters capital or not mixed with numbers as after  are allowed  <br>
- My Name -> False -> space not allowed <br>
- MyName -> Correct -> Capital and small characters together are allowed<br>
- #Name -> False -> special characters not allowed<br>
- $Name -> False -> same as answer before<br>
- N@me -> False -> special characters not allowed even not at first<br>
- int -> False->is a data type so its taken <br>
- float -> False -> same amswer as before<br>
