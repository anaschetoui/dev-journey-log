## Problem 09

>### Write a program to ask the user to enter:
> - Number1, Number 2, Number 3 <br>
>### Then Print the Sum of entered numbers<br>
>Example Inputs:<br>
>10<br>
>20<br>
>30<br>
>Outputs→<br>
>60<br> 

### Steps: 

**Step 1:** Read Num1<br>
**Step 2:** Read Num2<br>
**Step 3:** Read Num3<br>
**Step 4:** Result = Num1+Num2+Num3<br>
**Step 5:** Print Result<br>
